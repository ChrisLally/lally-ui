# @chris-lally/ui

CLI and distribution entrypoint for `@chris-lally/ui`.

## Install

```bash
pnpm add -D @chris-lally/ui
```

## Usage

```bash
lally-ui --help
```

Current commands:

- `lally-ui init`
- `lally-ui add <namespace/item>`
- `lally-ui registry connect [--url <registry-url>]`
- `lally-ui registry export --out public/r`
- `lally-ui doctor`

Examples:

- `lally-ui add fumadocs/sdk-layout`
- `lally-ui registry connect`
- `lally-ui registry export --out public/r`
